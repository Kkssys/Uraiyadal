# 💬 Uraiyadal - MERN Chat App

Real-time chat application with OTP verification, friend system, and media sharing.

## 🚀 Features
- ✅ Email OTP verification
- ✅ Real-time messaging (Socket.io)
- ✅ Friend requests system
- ✅ Block/Unblock users
- ✅ Media sharing (images, videos, files)
- ✅ Read receipts (seen ✓✓)
- ✅ Dark/Light mode
- ✅ Online/Offline status
- ✅ Typing indicators
- ✅ Profile pictures

## 🛠️ Tech
- **Frontend:** React, Socket.io-client, Axios
- **Backend:** Node.js, Express, MongoDB, Socket.io, JWT

## 📦 Installation

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (new terminal)
cd frontend
